# CICD 도입

#### 깃허브 액션 (hosted runner 활용)

- release 브랜치에 push 발생
  - 메인 서버의 경우 js로 빌드
- scp를 통해 배포 서버로 모듈 전달
  - RSA 키로 인증
- ssh로 배포 서버의 쉘 파일 실행
  - RSA 키로 인증

#### 젠킨스 (배포 서버에 위치)

- release 브랜치에 push 발생
- release 브랜치 clone
  - 메인 서버의 경우 js로 빌드
- 쉘 파일 실행
